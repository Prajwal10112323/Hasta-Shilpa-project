# Hasta-Shilpa

## Overview
Hasta-Shilpa is an Android application designed to support bamboo and cane artisans by connecting traditional craftsmanship with modern design trends.

## Problem Statement
Traditional artisans often continue producing old product patterns that do not perform well in modern urban markets. The project helps artisans access modern product ideas, blueprint measurements, and pricing guidance.

## Features
- Design Trend Feed
- Blueprint Measurement View
- Material Usage Tracker
- Price Suggestion System
- Marketplace Simulation
- Favorites and Saved Designs

## Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio
- Material Design 3
- Firebase (Planned)
- Room Database (Planned)

## Project Status
Prototype and UI workflow completed. Android build integration is currently under development.

## Goal
To modernize traditional bamboo and cane craftsmanship using digital tools and sustainable design support.
